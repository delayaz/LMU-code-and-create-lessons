Say something. The computer says it back.

First, get into Replit. Sign up with Google or GitHub — fastest option. Create a Python repl and name it yourname-day1. Then we write the first line of code any of us write today.

main.py
print("Hello, world!")
$ Hello, world!

Add a second line. Notice — two instructions, run top to bottom, in order. That's basically what a program is.

main.py
print("Hello, world!")
print("My name is Elin.")
$ Hello, world!
My name is Elin.
Your turn

Write 3 print() lines about yourself

Your name, your grade, and one fun fact. Run it. Make sure all three show up, in order, in the console.

  • Done fast? Add 2-3 more lines.
  • Feeling bold? Delete a quotation mark on purpose and see what the error says. Errors are normal — you'll see plenty.
!Common snag: forgetting the quotation marks around text, or starting with " and ending with '. Python is picky about matching them exactly.
Break
Burst 29:45 – 10:30

Give a value a name. Reuse it.

A variable is a labeled box that holds a value. Once it has a name, you don't retype the value — you just use the name.

main.py
name = "Elin"
grade = 10
favorite_subject = "Art"
gpa = 3.8

print(name)
print(grade)
$ Elin
10

Text is a string. Whole numbers are integers. Decimals are floats. Now drop variables straight into a sentence with an f-string — the f right before the quotes is what makes the curly braces work.

main.py
print(f"My name is {name} and I'm in grade {grade}.")
$ My name is Elin and I'm in grade 10.
Your turn

Build an "About Me" program

At least 4 variables: your name, your grade, and two more — favorite food, a hobby, your pet's name, whatever you want. Then print one sentence using an f-string that weaves all of them together.

!Common snags: forgetting the f before the quotes (the braces print literally instead of substituting), or using a variable name with a space in it — favorite food errors, favorite_food works.
Break
Burst 310:40 – 11:20

Now it talks back.

input() pauses your program and waits for a person to type something. Whatever they type gets stored — wherever you point it.

main.py
name = input("What's your name? ")
print(f"Nice to meet you, {name}!")
$ What's your name? Yutaro
Nice to meet you, Yutaro!
Your turn

Turn your About Me into an interactive greeter

Instead of hardcoding your own info, use input() to ask the user for their name and a couple facts about them — then print a personalized greeting back.

  • Example shape: ask for a name, a hobby, a favorite food → print one warm sentence using all three.
!Even though a grade is conceptually a number, input() always hands back text. That'll matter later this week — no need to fix it today.
