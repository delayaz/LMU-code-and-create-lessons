Basic math + putting numbers into sentences

Python does math the way you'd expect, using the + - * / operators, but input() always hands back text, even when it looks like a number. So before you can do math with it, you have to convert it.

Here's an example of a basic tip calculator:

main.py
bill = float(input("What's the bill total? "))
tip_percent = float(input("What percent tip? "))

tip_amount = bill * (tip_percent / 100)
total = bill + tip_amount

print(f"Tip: ${tip_amount:.2f}, Total: ${total:.2f}")
$ What's the bill total? 42
What percent tip? 18
Tip: $7.56, Total: $49.56

Two new things: calling float(...) turns text into a decimal number, and {tip_amount:.2f} inside an f-string rounds to exactly 2 decimal places — handy for money.

!Common snag: forgetting to wrap input() in float() or int() — without it, Python treats the numbers as text and + will glue them together instead of adding them ("4" + "2" becomes "42", not 6).

Comments — notes for humans, not the computer

Anything after a # is ignored by Python entirely. Comments exist purely so a human reading your code later — including you, next week — understands what's going on.

main.py
# Ask for the bill and tip percent
bill = float(input("What's the bill total? "))
tip_percent = float(input("What percent tip? "))

# Calculate the tip and the total
tip_amount = bill * (tip_percent / 100)
total = bill + tip_amount

# Show the result, rounded to cents
print(f"Tip: ${tip_amount:.2f}, Total: ${total:.2f}")
$ (same output as before — comments don't change behavior)
